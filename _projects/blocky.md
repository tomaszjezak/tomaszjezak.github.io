## Overview

**Blocky** is a custom data marketplace enabling AI companies to reduce **bias** by commissioning niche datasets from diverse contributors.

Built at [Sentium](https://sentium-d639f.web.app/) to address the growing need for diverse, high-quality training data in machine learning.

## Key Features

- **Marketplace Platform**: Connect AI companies with data contributors for custom dataset creation
- **Bias Reduction**: Focus on underrepresented demographics and edge cases
- **Smart Query Processing**: Created synthetic dataset to fine-tune **BLOOM-3b** for extracting keywords from company queries, enabling intelligent matching between data needs and available contributors

## Technical Stack

- Fine-tuned large language model (BLOOM-3b) for natural language understanding
- Marketplace infrastructure for data exchange
- Automated keyword extraction and matching system
