2024

  1. KDD Cup
    Academic Question Answering using Dense Retrieval with Supervised Contrastive Learning
    Tomasz Jezak
    2024
    Abs Bib PDF Video Code

    Developed a dense retrieval model for Academic Question Answering utilizing Supervised Contrastive (SupCon) loss — the first application of SupCon to text retrieval. Incorporated hard negative mining during training to enhance model robustness and implemented HNSW for efficient nearest-neighbor retrieval, achieving leaderboard score 0.119 (35th place).

  2. UCLA CS188
    Image Diffusion Fine-Tuning for Fan Art Generation
    Tomasz Jezak
    2024
    Abs Bib HTML

    Fine-tuned Stable Diffusion and SDXL with custom datasets to synthesize fan art. Developed a novel technique by hacking DreamBooth to store environments across diffusions instead of objects, enabling more creative and contextual image generation.
